# JavaScript 30 Projects Challenge

## Overview
A series of **30 diverse projects in 30 days** to strengthen JavaScript skills. Projects range from **games to tools and coding challenges**, providing hands-on practice across **all aspects of JavaScript**, including DOM manipulation, APIs, logic, and performance.

The goal of this repository is to build strong practical experience through daily hands-on projects rather than focusing only on theory.
